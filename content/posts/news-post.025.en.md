Brasília, July 14, 2026

**Ana Beatriz Norberto da Silva and Beatriz Vieira Nascimento defended the bachelor's thesis that gave rise to Git Ranking, a gamified application for managing and motivating agile teams integrated with GitHub.**

The thesis, titled _"Git Ranking: Developing a Gamified Application for Managing and Motivating Agile Teams as a GitHub App,"_ was supervised by Prof. {{< link-interno "/people/sergio_freitas" "Sergio Freitas" >}} and evaluated by a committee composed of Prof. Carla Silva Rocha Aguiar and Prof. Cristiane Soares Ramos.

Git Ranking was designed as a **GitHub App**, integrating elements of the **Octalysis** framework to balance teams' intrinsic and extrinsic motivators. The solution consumes **GitHub Webhooks**, processed by a **Node.js** backend and presented through a **React/Next.js** interface, offering a metrics dashboard, an achievements profile with badges, and continuous social feedback based on developers' actual actions in the repositories.

The project's source code is available on [GitHub](https://github.com/CedisUnB/GitRanking), and the application can be accessed at [gitranking.cedis.tec.br](https://gitranking.cedis.tec.br/).

Git Ranking now also joins CEDIS's showcase of **digital products** and can be found on the {{< link-interno "/products/gitranking" "Git Ranking page at CEDIS" >}}.
